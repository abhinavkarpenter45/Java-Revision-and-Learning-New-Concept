Q1. What is JDK?
Answer: JDK is a software development kit used to develop Java applications. 
It contains JRE, JVM, compiler, debugger, and other development tools.


Q2. What is JRE?
Answer: JRE provides the runtime environment required to execute Java applications. 
It contains JVM and Java libraries.


Q3. What is JVM?
Answer: JVM is a virtual machine that executes Java bytecode 
and converts it into machine code.


Q4. Why is Java Platform Independent?
Answer: Java programs are compiled into bytecode, 
which can run on any operating system having a compatible JVM.


Q5. Difference between JDK and JRE?
Answer:
    JDK is used for development.
    JRE is used for running Java applications.


Q6. Difference between JRE and JVM?
Answer:
    JRE provides the runtime environment.
    JVM is the engine that executes bytecode.


Q7. Can we run Java without JDK?
Answer: Yes. If the program is already compiled (.class file), 
it can be run using JRE (or a Java runtime). 
To write and compile Java code, JDK is required.


Q8. Which memory stores objects?
Answer: Heap Memory.


Q9. Which memory stores local variables?
Answer: Stack Memory.


Q10. What is Bytecode?
Answer: Bytecode is the intermediate code generated by the Java compiler (javac) 
from the Java source code. It is platform-independent and is executed by the JVM.
